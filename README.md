# Seaborn Data Visualization

This repository contains Python programs and projects using the Seaborn library to create advanced statistical visualizations.

## 📌 Features
- Distribution plots (distplot, histplot, kdeplot)
- Categorical plots (barplot, countplot, boxplot)
- Relational plots (scatterplot, lineplot)
- Heatmaps and correlation matrices
- Pairplots for multivariate analysis
- Beautiful and informative statistical graphics

## 🛠 Technologies Used
- Python
- Seaborn
- Matplotlib
- Pandas
- NumPy

## 📂 Project Structure
- Basic plots
- Statistical visualizations
- Correlation analysis
- Real-world datasets
- Practice programs

## 🎯 Purpose
This project is designed to learn and demonstrate statistical data visualization techniques using Seaborn for better data insights.
